# editkey の使い方

`editkey` は、ログインしていない状態で投稿した作品をあとから編集するためのキーです。
また、作品の公開設定によっては、作品を閲覧するためのキーとしても使われます。

## 共通の注意

- `editkey` は投稿時に自分で決める任意の文字列です。
- 忘れると、ログインなしで投稿した作品を編集できなくなります。
- 他人に知られると、その人が匿名投稿を編集したり、非公開作品を閲覧したりできる場合があります。
- キーを設定した場合は、作品の URL と一緒に安全な場所へ保管してください。
- 保存フォームでは `editkey` の入力内容がブラウザの `localStorage` に保存されます。共有パソコンでは注意してください。

## ログインなしで投稿する場合

### 編集に使う

ログインなしで新規投稿するときは、保存フォームの「編集キー」欄にキーを入力します。
保存後、同じ作品の編集ページを開いて、同じ `editkey` を入力して保存すると作品を更新できます。

匿名投稿にはログインユーザーのような所有者情報がないため、編集時は保存済みの `editkey` と入力値が一致する必要があります。
キーが空のまま投稿した場合は、空のキーでも一致するため、実質的に誰でも編集できる状態になります。

### 閲覧に使う

- 公開作品: `editkey` は不要です。
- 非公開作品 (`is_private=1`): 保存済みの `editkey` が一致した場合だけ閲覧できます。
- 限定公開 (`is_private=2`): ログインなしの新規投稿では選択できません。

匿名投稿の非公開作品を閲覧するときは、表示されたキー入力画面に `editkey` を入力します。

## ログインして投稿する場合

### 編集に使う

ログインして投稿した作品は、投稿者本人または管理者であれば `editkey` なしで編集できます。
他のログインユーザーが編集することはできません。

`editkey` はログイン投稿の編集認証には使われません。

### 限定公開に使う

公開設定を「限定的に公開」(`is_private=2`) にし、`editkey` を設定すると、キーを知っている人にだけ作品を閲覧してもらえます。

- 投稿者本人と管理者はキーなしで閲覧できます。
- それ以外の人は、正しい `editkey` の入力が必要です。
- 限定公開作品は、通常の作品一覧には表示されません。

### 非公開との違い

「非公開」(`is_private=1`) は投稿者本人と管理者だけが閲覧できます。
非公開作品では、他の人が `editkey` を知っていても閲覧できません。

## 設定のまとめ

| ログイン状態 | 公開設定 | `editkey` の主な用途 |
| --- | --- | --- |
| ログインなし | 公開 | 匿名投稿をあとから編集するためのキー |
| ログインなし | 非公開 | 編集キー、および閲覧キー |
| ログインなし | 限定公開 | 新規投稿では選択不可 |
| ログインあり | 公開 | 基本的に不要。設定した場合も編集権限はアカウントで判定 |
| ログインあり | 非公開 | 閲覧権限には使われない。本人と管理者だけ閲覧可能 |
| ログインあり | 限定公開 | 他の人に閲覧を許可するための共有キー |
